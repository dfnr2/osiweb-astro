The OSI UK User Group Newsletter was published by the British user group community for OSI (Ohio Scientific) computer enthusiasts in the United Kingdom. The newsletter provided programming tips, hardware modifications, system notes, and community information specifically for UK OSI users.

*Many thanks to Pete Targett for these scans!*
